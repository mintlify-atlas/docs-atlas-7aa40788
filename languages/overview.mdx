## Common Patterns

Across all languages, you'll find these recurring patterns:

### Infinite Loops

Most implementations use an infinite loop to continuously print 'e':

<CodeGroup>
```c C
for(;;) {
    printf("e");
}
```

```python Python
while True:
    print('e', end='')
```

```go Go
for {
    print("e")
}
```
</CodeGroup>

### Recursion

Functional languages often prefer recursive approaches:

<CodeGroup>
```haskell Haskell
main = putStr $ cycle "e"
```

```scheme Scheme
(define (e)
    (display "e")
    (e))
(e)
```

```ocaml OCaml
let rec eee e = Printf.printf "e"; eee ();;
let () = eee ();;
```
</CodeGroup>

### System Calls

Assembly implementations directly invoke system calls for output:

```asm x86-64 Linux
mov rax, 1      ; sys_write
mov rdi, 1      ; stdout
mov rsi, msg    ; buffer
mov rdx, 1      ; length
syscall
```
