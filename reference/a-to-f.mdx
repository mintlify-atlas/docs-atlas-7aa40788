### Platform-Specific Assembly Variants

- **DOS Assembly** (e_dos.asm) - 16-bit DOS implementation
- **iOS ARM32** (e_iOS_arm32.S) - 32-bit ARM for iOS
- **iOS ARM64** (e_iOS_arm64.S) - 64-bit ARM for iOS  
- **LC-3** (e_lc3.asm) - Little Computer 3 educational ISA
- **Linux ARM** (e.linux.arm.S) - ARM Linux implementation
- **Linux RISC-V 64** (e_linux_riscv64.S) - RISC-V 64-bit Linux
- **macOS 32-bit** (e_macOS_32.S) - Intel 32-bit macOS
- **macOS 64-bit** (e_macOS_64.S) - Intel 64-bit macOS
- **Master Boot Record** (e.mbr.asm) - Bootable MBR implementation
- **MIPS** (e_mips.asm) - MIPS architecture
- **Motorola 68000** (e.x68) - 68k assembly
- **Standalone** (e_standalone.asm) - No OS dependencies
- **Windows 32-bit** (e_win32.asm) - Win32 implementation
- **Windows 64-bit** (e_win64.asm) - Win64 implementation
